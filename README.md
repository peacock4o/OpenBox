# OpenBox

An Obsidian notebook linking together boxing learning materials and concepts. Primarily populated with materials from Frolov boxing. Soviet style focus.

![OpenBox demonstration](img/readme.gif)

## Requirements
- Obsidian
- Templater plugin for Obsidian

## Installation
- Open this fodler as an Obsidian notebook
- Ensure Templater is installed
- Add the two files in `_templates` as startup scripts, under Settings -> Templater -> Startup Templates -> Add New Startup Template
- Close and reopen to run the startup scripts
